# Calls
./dot   init        ~/meyerk                  ; init project folder, copy "default" theme
./dot   article     ~/meyerk articleName      ; add article with date as folder name, articleName as filename
./dot   page        ~/meyerk pageName         ; add page with pageName as folder name, pageName as filename
./dot   build       ~/meyerk                  ; (re-)create "rendered"
./dot   theme       ~/meyerk newTheme         ; copy theme from themes folder in dot/themes, forces build

Features:
- create a scaled down version of images and wrap images in a "picture" set
- what about custom tags (see CBM)
